Marcus Sync

Reminders → Markdown

macOS 13+ · Menu Bar App

What It Does

Marcus Sync lives in your menu bar and syncs Apple Reminders to any Markdown file. Built for Obsidian users who want their tasks in their vault without manual copy-paste.

One-way, additive sync. Your manual edits are preserved — Marcus only adds new items, never overwrites what you've written.

Features

Menu Bar Only

No dock icon, no window clutter. Lives quietly in your menu bar.

Any Markdown File

Point it at any .md file — Obsidian vault, Logseq, or plain text notes.

Preserves Manual Edits

Additive sync only. Your hand-written notes stay untouched.

Two Output Formats

Emoji checkboxes or Dataview-compatible format for advanced queries.

Smart Validation

Checks your settings before syncing. Blocking issues prevent sync, warnings let you decide.

Native macOS

Built with SwiftUI and EventKit. No Electron, no web views.

Output Formats

Emoji Format

## Reminders
✅ Buy groceries
🟡 Schedule dentist appointment
🟡 Review quarterly report

Dataview Format

## Reminders
- [x] Buy groceries [due:: 2025-01-15] [completed:: 2025-01-14]
- [ ] Schedule dentist appointment [due:: 2025-01-20]
- [ ] Review quarterly report [due:: 2025-01-25]
